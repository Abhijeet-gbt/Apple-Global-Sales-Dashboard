🍎 Apple Global Sales Dashboard

An interactive multi-page sales analytics dashboard built with Streamlit

Analyzing 11,500+ Apple product sales product sales records across 47 countries (2022-2024)

Covers Revenue Trends city wise performance ,product category insights with region wise performance


## Dashboard 📸 Preview 
[Overview](/screenshots)

## 🛠️ Tech Stack
- Python
- Pandas — data manipulation
- Matplotlib & Seaborn — visualizations
- Streamlit — dashboard framework

## ✨ Features
- Overview — total revenue and units sold at a glance
- Sales — top 5 cities by revenue for any country
- Trends — month-wise revenue trends across years
- Products — category-wise units sold by region
- Gallery — discount vs units sold relationship by category
